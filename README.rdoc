==== Travel Flower =====

Travel Flower is the group effort of the Dev Bootcamp Coyotes, Team 3, Phase 3 pre-final project weekend.  

The app is modeled off of Quora / Stack Overflow and allows users to post questions regarding travel.  Users can also answer and upvote other answers.
